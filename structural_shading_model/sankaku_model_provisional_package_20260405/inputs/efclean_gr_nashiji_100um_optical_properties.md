# エフクリーンGRナシジ 100μm 屋根面光学特性 暫定パラメータ

## 1. 前提
- 対象: **エフクリーン GRナシジ 100μm**
- 用途: 温室屋根面の **直達透過 / 散乱化 / 波長帯域平均透過率** を光遮蔽モデルへ入れるための暫定値
- 注意:
  - **GRナシジ100μm 固有の入射角別透過率の公開実測表は未確認**
  - 角度依存は **AGC公開のエフクリーン自然光100μm の角度特性**で代用
  - 分光透過率は **AGC公開グラフの目視近似**
  - 数値は**保証値ではない**

## 2. 公開資料から直接使う値
- 材質: **ETFE**
- ナシジシリーズの散乱光割合: **約45%**
- 結露時: **直達光の割合が増加**
- GRナシジ100μm の **300 nm透過率: 21%**

## 3. 乾燥時の暫定代表値
- **PAR全透過率**: `0.903`
- **直達成分比**: `0.55`
- **散乱成分比**: `0.45`
- **天空散乱光透過率**: `0.90`

### 乾燥時の分解
- `T_par_total = 0.903`
- `T_par_dir = 0.903 × 0.55 = 0.497`
- `T_par_dif_from_beam = 0.903 × 0.45 = 0.406`

## 4. 結露時の暫定代表値
公開資料は定性的記述のみなので、感度解析用の仮定値とする。

- **PAR全透過率**: `0.90`
- **直達成分比**: `0.75`
- **散乱成分比**: `0.25`

### 結露時の分解
- `T_par_total_cond = 0.90`
- `T_par_dir_cond = 0.90 × 0.75 = 0.675`
- `T_par_dif_from_beam_cond = 0.90 × 0.25 = 0.225`

## 6. 波長積分後の代表透過率
重み付けは **ASTM G-173 / AM1.5 標準太陽スペクトル**を用いる。

### 積分結果
- **PAR 400–700 nm 光子重み付き透過率**: `0.903`
- **PAR 400–700 nm エネルギー重み付き透過率**: `0.901`
- **UVA 320–400 nm エネルギー重み付き透過率**: `0.698`
- **短波 320–2500 nm エネルギー重み付き透過率**: `0.903`
- **短波 305–2800 nm エネルギー重み付き透過率**: `0.902`

## 8. モデル投入式
### 乾燥時
- `T_par_total(θ) = 0.903 × K(θ)`
- `T_par_dir(θ) = 0.55 × T_par_total(θ)`
- `T_par_dif_from_beam(θ) = 0.45 × T_par_total(θ)`

### 結露時
- `T_par_total_cond(θ) = 0.90 × K(θ)`
- `T_par_dir_cond(θ) = 0.75 × T_par_total_cond(θ)`
- `T_par_dif_from_beam_cond(θ) = 0.25 × T_par_total_cond(θ)`

### 天空散乱光
- `T_sky_diffuse = 0.90`
